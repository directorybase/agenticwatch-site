## Known issues / lessons learned

### update-stars workflow: stars.json ignored by git (FIXED 2026-05-17)
- **Root cause:** `data/stars.json` is listed in `.gitignore` (with comment "Generated by GitHub Actions — do not commit from local"). The workflow's `git add data/stars.json` failed with exit code 1 because git refuses to stage ignored files without `-f`.
- **Fix:** Changed `git add data/stars.json` → `git add -f data/stars.json` in `.github/workflows/update-stars.yml` (commit f66d306).
- **Intent preserved:** `.gitignore` entry stays — prevents accidental local commits. `-f` in the action is the correct pattern for action-managed generated files.

### Deprecation warning (active)
- `actions/checkout@v4` runs on Node.js 20, which GitHub is deprecating. Forced to Node.js 24 default starting 2026-06-02. Not causing failures yet — upgrade when convenient.
